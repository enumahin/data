# Data Package

## Description

This is a data package for the CDR project.

## Requirements

- Java 17
- Maven

## Installation

1. Clone the repository
2. Run `mvn install`

## Usage

1. Run `mvn package`
2. Include the generated jar file or the github link in your project
3. Use the `com.alienworkspace.cdr.model` package
